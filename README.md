# NLW-Operator

#### Eu finalizei dois cursos do último NLW, sendo eles da trilha Iniciante e Fullstack

## Sobre os projetos

### Trilha Iniciante
A trilha iniciante aplicou na prática como implementar a função (do Cloudinary) de um agente de uma API do Claude.
Eu usei ferramentas como: Javascript, uso de Api e Cloudinary.

### Trilha Fullstack

A trilha fullstck aplicou na prática como trabalhar com um agente e seus mcps, para um projeto que consiste na avaliação de códigos de maneira cômica.
Eu usei ferramentas como: React.js, Tailwind.css, Next.js, Node.js, Opencode, Biome.js, PostgreSQL e Drizzle.
MCPs: Superpowers, Plawright, Pencil, Polar entre outros.

## Experiência

Ambas trilhas trabalharam com a questão do uso de AI pra facilitar processos, visto que o mercado de trabalho visa cada vez mais o uso dessas ferramentas e saber manipulá-las.
